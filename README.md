
# ScoreHub: Система автоматизации записи и оценивания учебных достижений

### Обзор проекта
**ScoreHub** — это информационная система для управления процессом защиты учебных работ («дорешек»). Проект решает проблему очередей, отсутствия прозрачности в оценивании и ручного контроля лимитов попыток. 

Система интегрирована с учебным расписанием **TSU InTime** и обеспечивает автоматический контроль бизнес-правил курса.

### Технологический стек
* **Database:** PostgreSQL.
* **Integration:** SOAP (TSU InTime API), Push Notifications (FCM).
* **Tools:** Draw.io (UML), Notion (RBAC Matrix), ELK Stack (Logging).

### Артефакты проектирования

#### 1. Управление ролями и доступом (RBAC)
Спроектирована матрица полномочий для трех ключевых акторов: **Студент**, **Ассистент**, **Преподаватель**. 
> *Посмотреть полную матрицу можно в разделе `/docs/permissions-matrix.md`.*

#### 2. Бизнес-процессы (BPMN)
Описан сквозной процесс записи на дорешку, включая:
* Проверку лимита попыток (не более 2-х в семестр).
* Синхронизацию слотов с внешним API.
* Обработку негативных сценариев (ошибки API, отсутствие свободных мест).

#### 3. Архитектура данных (ERD & Class Diagram)
* **ER-диаграмма:** Спроектирована реляционная структура для хранения данных о сессиях, командах и оценках.
* **UML Class Diagram:** Реализована объектно-ориентированная модель с наследованием ролей от базового класса `User` и логикой пересчета баллов (`weight check`).

#### 4. Интеграции и последовательность (Sequence Diagram)
Детально прописано взаимодействие между фронтендом, бэкендом и внешними сервисами:
* Авторизация через JWT.
* Получение данных о занятиях через SOAP Envelope.
* Отправка Push-уведомлений ассистентам при вызове студентом.

### 📋 Сценарии использования (User Stories)
* **Студент:** «Как студент, я хочу записаться на слот через интерфейс, чтобы не стоять в живой очереди».
* **Ассистент:** «Как ассистент, я хочу получать уведомление о готовности студента, чтобы эффективно распределять время на паре».
* **Преподаватель:** «Как преподаватель, я хочу иметь возможность пересмотреть оценку, чтобы разрешать спорные ситуации».
